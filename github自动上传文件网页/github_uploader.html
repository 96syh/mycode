FAQ

常见 问题 解答

遇到问题？这里有你需要的答案

"Resource not accessible" 错误

这是最常见的错误！表示 Token 没有写入权限。

解决方法：

1. 删除当前的 Token

2. 重新创建一个 Token

3. 必须勾选完整的 "repo" 权限

"Not Found" 错误

最常见的原因：

• 分支名称错误（新仓库默认是 "main"）

• 仓库名称或所有者填写错误

• Token 没有 repo 权限

如何创建正确权限的 Token？

1. 访问 GitHub Token 设置

2. 点击 "Generate new token (classic)"

3. 勾选完整的 "repo" 权限

4. 点击底部 "Generate token"

5. 立即复制 Token（只显示一次）

浏览器不支持
